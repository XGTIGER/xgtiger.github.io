ansible 快速上手--Prompts

五 03 四月 2015
By wyk

In Ansible.

prompt关键字能够实现交互式动态提供变量给playbook * 常规用法 

---
- hosts: all
  remote_user: root

  vars:
    from: "camelot"

  vars_prompt:
    - name: "name"
      prompt: "what is your name?"
    - name: "quest"
      prompt: "what is your quest?"
    - name: "favcolor"
      prompt: "what is your favorite color?"
  • 使用 default:
vars_prompt:

  - name: "release_version"
    prompt: "Product release version"
    default: "1.0"
  • use private
vars_prompt:

  - name: "some_password"
    prompt: "Enter password"
    private: yes

  - name: "release_version"
    prompt: "Product release version"
    private: no
  • If Passlib is installed, vars_prompt can also crypt the entered value so you can use it, for instance, with the user module to define a password,You can use any crypt scheme supported by ‘Passlib’:
vars_prompt:

  - name: "my_password2"
    prompt: "Enter password2"
    private: yes
    encrypt: "sha512_crypt"
    confirm: yes
    salt_size: 7
